LED SmartClock

Web Installer

Install LED SmartClock on a blank ESP32 over USB without setting up PlatformIO. This installer flashes the full initial image for release __TAG__.

Download firmware.zip Download update.bin

What This Installs

The installer flashes the merged firmware.bin image for first-time installs. After the clock is installed, future updates should use update.bin from the clock's own web interface.

  • Initial install: firmware.bin via this page
  • OTA updates later: update.bin via the clock web UI
  • Release package: firmware.zip contains both files and the local installer

Before You Start

  • Use Google Chrome or Microsoft Edge on desktop.
  • Connect the ESP32 over USB with a data cable.
  • Close any serial monitor or other program using the port.
  • If auto-reset fails, hold BOOT while starting the install.

After Flashing

  • Wait for the clock to reboot.
  • Join the LEDSMARTCLOCK setup AP if this is a fresh install.
  • Use onboarding to set Wi-Fi, password protection, and API keys.
  • Use the built-in OTA page for later updates with update.bin.

Manual Recovery

If browser flashing is unavailable, download firmware.zip and flash over USB using the instructions in the project README.
